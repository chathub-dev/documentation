ChatHub 有两种会员类型：[**订阅**](/membership/subscription) 和 [**终身**](/membership/lifetime)。

|                                    | 订阅 (Pro+/Unlimited）                    | 终身               |
| ---------------------------------- | ---------------------------------------- | ----------------- |
| Access to ChatHub Premium features | Yes                                      | Yes               |
| Requires own API key or accounts   | No                                       | Yes               |
| Payment model                      | Monthly or yearly payment                | One-time payment  |
| Works with                         | ChatHub extension & webapp & mobile apps | Only ChatHub extension |
| Web Access                         | Yes                                      | Yes               |
| Summarize Chat 										 | Yes                                      | Yes               |
| Code Preview                       | Yes                                      | Yes               |
| Document Upload                    | Yes                                      | No                |
| Image Generation                   | Yes                                      | No                |
| Custom Chatbots                    | No                                       | Yes               |

## FAQ

### 我应该选择哪种会员？

订阅计划提供即用的 AI 服务，简单且适合大多数人。

如果你知道如何获取和使用 API key，你也可以选择终身会员。

### "Requires own API key or accounts" 是什么意思？

终身会员授予你访问 ChatHub 的高级功能（如 Web 访问和自定义聊天机器人），但你需要提供自己的 API 密钥或账户来使用 AI 服务。你可能需要为这些服务分别付费。

- API keys: OpenAI keys, Anthropic keys, Gemini keys等
- 账户: ChatGPT 账户, Claude 账户, Google 账户等
