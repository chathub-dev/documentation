The browser extension is suitable for users who want to use their own API key or create [custom chatbots](/custom-chatbots/). You can install it from the [**Chrome Web Store**](https://chathub.gg/chrome).

In the browser extension, we offer you three options: use our AI service, use your own account, or use your own API Key：
