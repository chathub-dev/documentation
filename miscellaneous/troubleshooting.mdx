## "This license key has reached the activation limit"

This error message indicates that your ChatHub license key has been activated on the maximum number of devices (browsers) permitted by your license. ChatHub licenses have varying activation limits, which are specified before purchase.

You have two options to resolve this issue:

<AccordionGroup>
  <Accordion title="Upgrade Your License">
		Upgrade your license to increase the number of devices you can activate. You can do this by contacting [ChatHub support](/miscellaneous/contact) to pay the price difference.
	</Accordion>
	<Accordion title="Use the License Manager Tool">
	<Steps>
		<Step title="Open the License Manager tool">
			[https://app.chathub.gg/license-manager](https://app.chathub.gg/license-manager)
		</Step>
		<Step title="Enter your license key">
			View a list of all currently activated devices (browsers).
		</Step>
		<Step title="Remove one or more devices from the list">
			This will free up activation slots for this license key.
		</Step>
		<Step title="Activate your current device">
			Once a slot is available, you can activate your current device in the ChatHub browser extension.
		</Step>
	</Steps>
	</Accordion>
</AccordionGroup>

## "You have sent too many messages to the model. Please try again later."

`{"detail":{"message":"You have sent too many messages to the model. Please try again later.","code":"model_cap_exceeded","clears_in":0}}`

This error was caused by ChatGPT. Since you're using a ChatGPT account in ChatHub browser extension, the same usage restrictions apply (in simple terms, you will see the same error on chatgpt.com).

To resolve this issue, you can:

- Wait for the ChatGPT usage limit to reset.
- Set OpenAI API key in ChatHub browser extension.
- Subscribe to ChatHub Pro+ or Unlimited plan.
