title: Image Generation
---

There are two ways to generate images in ChatHub:

1. Use the [Image Generator](https://app.chathub.gg/image-generator) tool
2. Ask models to generate images in chat

## Image Generator

The [Image Generator](https://app.chathub.gg/image-generator) is a tool that allows you to generate images with various image models.

Supported models:

- Flux.1
- Fast SDXL
- Dall-E 3
- GPT-4o (supports image editing)


![Image Generator](/images/image-generation/generator.png)


## Image Generation in Chat

You can ask models to generate images in chat while chatting with them.
