Although ChatHub is free to use, you can purchase a Premium membership to unlock premium features.

There are two types of Premium membership: [**Subscription**](https://doc.chathub.gg/membership/subscription) and [**Lifetime**](https://doc.chathub.gg/membership/subscription).

|                                    | Lifetime          | Subscription (Pro+/Unlimited）            |
| ---------------------------------- | ----------------- | ---------------------------------------- |
| Works with                         | ChatHub extension | ChatHub extension & webapp & mobile apps |
| Access to ChatHub Premium features | Yes               | Yes                                      |
| Requires own API key or accounts   | Yes               | No                                       |
| Payment model                      | One-time payment  | Monthly or yearly payment                |
| Document Upload                    | No                | Yes                                      |
